## Emacs

### Dirtree

* Usage
  * `M-l` -- eproject-dirtree, will open dirtree with the root at the
  current eproject.
  * `M-x dirtree` -- Will open the dirtree buffer
  * `e` -- Expand node
  * `u` -- Up to parent
  * `r` -- Up to root?
